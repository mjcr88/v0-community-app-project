### ğŸ“… Events & Calendar

- **Feature**: **RSVP to Series**
  A new streamlined modal allows you to RSVP to "This Event Only" or "This and All Future Events" in a series, making it easier to commit to recurring activities like weekly fitness classes.
- **Feature**: **Location Conflict Detection**
  Admins now receive an immediate warning if they attempt to book a space that is already occupied by another event at the same time, preventing double-bookings.
- **Feature**: **Location Filter**
  A new dashboard filter allows residents to quickly find all events happening at a specific venue, such as the "Community Center" or "Poolside".
- **Fix**: **Accurate Widget Counts**
  The "Upcoming Events" widget now correctly calculates attendee numbers for individual occurrences of a series, rather than showing a generic series-wide count.

### ğŸ‘¥ Residents & Directory

- **Feature**: **Smart Family Link**
  When admins add a new resident to a lot, the system now automatically detects existing families on that property and offers to link them, keeping household data clean and organized.
- **Improvement**: **Real-Time Check-In Sync**
  Marking yourself as "Going" or "Checked In" now instantly updates across the Dashboard, Priority Feed, and Map view without needing to refresh the page.
- **Fix**: **Directory Tab Alignment**
  Resolved a visual issue where tabs in the Neighbor Directory were misaligned on desktop screens. The new grid layout ensures perfect symmetry and better touch targets on mobile.
- **Fix**: **Search Reliability**
  Improved the resident search function to correctly handle partial names and case sensitivity, ensuring you can find neighbors even if you only remember part of their name.

### ğŸ—ºï¸ Interactive Map

- **Feature**: **User Location Beacon**
  A new "Blue Dot" indicator shows your real-time location on the community map. Click the "Find Me" button to instantly center the view on yourself, making navigation easier.
- **Feature**: **Custom Map Colors**
  Admins can now assign specific colors to different paths and zones (e.g., blue for streams, green for trails) when uploading GeoJSON data, improving map readability.
- **Fix**: **Reliable GeoJSON Import**
  Fixed an issue where uploading complex map data (like trails) would merge separate paths into a single shape. Imports now strictly preserve the original geometry and elevation data.
